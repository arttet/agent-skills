# Skills

[Agent Skills](https://agentskills.io/) is an open format for adding specialized knowledge and workflows to AI agents.
Available skills in this repository. Open a skill to see how to invoke it.

| Skill                                      | What it does                                                                                                      |
| ------------------------------------------ | ----------------------------------------------------------------------------------------------------------------- |
| [Git Commit Planner](./git-commit-planner) | Analyzes changes and branch context, splits work into signed Conventional Commits, and outputs a copy-paste plan. |
